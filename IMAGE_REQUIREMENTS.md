# Image Requirements for Bring The Menu Landing Page

## Product Overview
Bring The Menu is an e-menu solution for restaurants where:
- Restaurants register and add their menu
- They generate QR codes for each table
- Customers scan the QR code
- Orders automatically include the table number
- Orders appear live on the restaurant dashboard
